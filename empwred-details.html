emPWRed

  • Category: Personal organizer tool
  • Built with: Python, Django, PostgreSQL, ZenQuotes API, Chart.js
  • Deployed URL: https://empwred.herokuapp.com/
  • GitHub URL: https://github.com/ripleymay/empwred

A group project between classmates Paolo (P), Will (W), and myself (R), this is a tool for people to track their progress on their goals while staying motivated to achieve balance in their lives.

I learned a lot about Git workflow (making pull requests, solving merge conflicts, etc.) through building this web app.
